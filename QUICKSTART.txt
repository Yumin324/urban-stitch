====================================
TROUBLESHOOTING
====================================

Problem: Can't access website
Solution: 
- Ensure XAMPP Apache is running
- Check URL: http://localhost/Urban%20Stitch/

Problem: Database connection error
Solution:
- Ensure MySQL is running in XAMPP
- Verify database name is "urbanstitch_store"
- Check config/database.php settings

Problem: Images not loading
Solution:
- Check internet connection
- Unsplash URLs require active internet

Problem: Can't login to admin
Solution:
- Use credentials: admin / admin123
- Clear browser cache
- Check session in browser

Problem: Cart not working
Solution:
- Enable localStorage in browser
- Clear browser cache
- Check browser console for errors

====================================
CUSTOMIZATION TIPS
====================================

1. Change Colors:
   - Edit CSS variables in assets/css/style.css
   - Primary color: --primary-color
   - Secondary color: --secondary-color

2. Add More Products:
   - Login to admin panel
   - Go to Products
   - Click "Add New Product"
   - Use Unsplash URLs for images

3. Modify Shipping Cost:
   - Edit assets/js/main.js
   - Change shipping value (currently 500)

4. Update Contact Info:
   - Edit includes/footer.php
   - Edit contact.php

5. Change Admin Password:
   - Login to phpMyAdmin
   - Browse admins table
   - Update password (use password_hash)
